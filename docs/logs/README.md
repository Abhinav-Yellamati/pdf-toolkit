# Logs

This folder contains human-readable development logs for demo explanation.

## Files

- `development-log.md`: High-level implementation timeline.
- `networking-fixes.md`: LAN and mobile networking fixes.
- `expo-fixes.md`: Expo startup and native compatibility fixes.
- `backend-fixes.md`: FastAPI and CORS updates.
- `ui-improvements.md`: Web and mobile UI polish summary.
- `feature-additions.md`: Implemented feature history.

These are documentation logs, not runtime logs. Runtime logs such as `backend-lan.log` and `expo-mobile.log` are ignored by git.
