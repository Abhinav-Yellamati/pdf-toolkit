# Project Summary

PDF Toolkit is a full-stack PDF processing platform built for web and mobile use. It allows users to upload documents, run common PDF operations, and download or share processed output files.

## Objective

The project demonstrates a practical document automation system with a clean user interface, reusable API architecture, and real PDF processing features.

## Key Highlights

- One FastAPI backend powers both web and mobile clients.
- React provides a responsive web dashboard.
- Expo React Native provides a physical-phone mobile workflow.
- PyMuPDF and pypdf perform real document manipulation.
- LAN networking is configured for Expo Go demonstrations.

## Demo Value

The project is easy to explain because each layer has a clear role: clients manage interaction, the API manages validation and routing, and the service layer manages PDF processing.
