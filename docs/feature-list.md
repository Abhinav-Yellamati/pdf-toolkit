# Feature List

## PDF Tools

- Compress PDF with image quality control.
- Merge multiple PDFs into one file.
- Split PDF into page files or custom ranges.
- Rearrange pages using a comma-separated page order.
- Convert PDF text into DOCX.
- Convert PDF pages into PNG images inside a ZIP file.
- Convert selected images into a PDF.
- Add a text watermark to every PDF page.
- Protect a PDF with password encryption.

## Platform Features

- Web UI and mobile UI share the same backend.
- Upload validation before API submission.
- Progress indicators during processing.
- Download result cards on web.
- Share/save support on mobile.
- Error and success toast messages.
- Health endpoint for backend checks.
- LAN-ready mobile configuration.
