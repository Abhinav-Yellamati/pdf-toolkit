# Demo Flow

1. Open the PDF Toolkit web app or Expo mobile app.
2. Show the available PDF tools.
3. Select a simple operation such as compress or merge.
4. Upload one or more files.
5. Submit the request and explain that the client sends multipart form data to FastAPI.
6. Show the downloaded result.
7. Briefly show backend routes and service functions.
8. Explain mobile LAN configuration and why physical phones need the computer IP address.
9. Open documentation files to show changelog, debugging timeline, testing report, and architecture notes.
