# Features

- Compress PDF files with bounded quality settings.
- Merge multiple PDF files into one document.
- Split a PDF by page or page ranges.
- Rearrange PDF pages using a custom order.
- Convert PDF text into a Word-compatible `.docx` file.
- Convert PDF pages into image files inside a zip archive.
- Convert multiple images into a single PDF.
- Add text watermarking to PDF pages.
- Protect PDF files with a password.
- Use the toolkit from both web and mobile clients.
