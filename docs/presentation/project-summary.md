# Project Summary

PDF Toolkit is a full-stack document processing application. It provides common PDF operations through a React web app, an Expo React Native mobile app, and a FastAPI backend.

The project is designed to show a complete software engineering flow: user interface, mobile support, HTTP APIs, backend validation, PDF processing, file downloads, debugging history, and documentation.

Core tools include PDF compression, merging, splitting, page rearrangement, PDF-to-Word conversion, PDF-to-image conversion, image-to-PDF conversion, watermarking, and password protection.
