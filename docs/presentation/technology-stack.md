# Technology Stack

## Frontend

- React for the web interface.
- Expo React Native for the mobile interface.
- JavaScript for client-side logic and API requests.

## Backend

- FastAPI for HTTP APIs.
- Starlette background tasks for cleanup after downloads.
- PyMuPDF for rendering, compression, image conversion, and watermarking.
- pypdf for merge, split, rearrange, and password protection workflows.

## Tooling

- npm for web and mobile dependency management.
- Python virtual environment for backend dependencies.
- Markdown logs and documentation for project traceability.
